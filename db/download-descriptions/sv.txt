freebsd5-i386-system:
    Hostad version för FreeBSD/i386 5.x. Notera att det inte
    kommer att fungera på FreeBSD 4.x-system.
freebsd-i386-system:
    Hostad version för FreeBSD/i386 5.x. Notera att det inte
    kommer att fungera på FreeBSD 4.x-system.
linux-i386-system:
    Hostad version för Linux/i386. Det här är den mest
    kompletta och mogna versionen, vilket menas att du kan
    köra alla inkluderade program och demon med den. Det här
    arkivet innehåller alla AROS system binaries kompilerade från
    subversions-trädet.
mingw32-x86_64-system:
    The hosted version for Microsoft Windows on x86_64 platform.
mingw32-i386-system:
    The hosted version for Microsoft Windows on i386 platform. Developed
    and compiled under Windows XP, but should work on all 32-bit Windows
    versions starting from Windows 98.
linux-x86_64-system:
    Hostad version för Linux/x86_64. Det här är den mest
    kompletta och mogna versionen, vilket menas att du kan
    köra alla inkluderade program och demon med den. Det här
    arkivet innehåller alla AROS system binaries kompilerade från
    subversions-trädet.
linux-ppc-system:
    Hostad version för Linux/PowerPC. Det här arkivet innehåller alla
    AROS system binaries kompilerade från subversions-trädet.
pc-i386-boot-floppy:
    Native version för i386-baserade PC AT datorer och kompatibla.
    Det här arkivet innehåller en startdiskett-image som du snabbt
    kan skriva till diskett och starta upp AROS med.
pc-i386-boot-iso:
    Native version för i386-baserade PC AT datorer och kompatibla.
    Detta är en bootbar CDROM ISO image, som innehåller alla
    AROS-system och bidragande program.
raspi-armhf-system:
    The native version for ARMv6 based Raspberry Pi computers.
    Contains all files needed to boot from a FAT formatted sd-card.
raspi-armhf-contrib:
    This archive contains all the contributed programs,
    compiled for the ARM architecture, for use with the
    RasPi/ARM port (using hard-float ABI). Please note that this
    version requires at least an ARMv6 CPU with VFP unit.
raspi-armhf-ports:
    This archive contains applications and games which are built
    from external sources for the armhf architecture (both hosted
    and native). Because of its size the archive will only be updated
    from time to time on the snapshots page.
pc-i386-contrib:
    Detta arkiv innehåller alla bidragande program komplierade
    för i386-arkitekturen. Dom fungerar både i (i386-linux)
    och native (i386-pc) versioner.
pc-x86_64-boot-iso:
    The native version for 64-bit (x86_64, amd64) PC computers.
    This is the bootable CDROM ISO image, which
    contains all AROS system and contributed programs.
pc-x86_64-smp-boot-iso:
    This is a highly experimental SMP build for pc-x86_64.
    Only recommended for Developers. This is a bootable CD-ROM ISO image.
pc-x86_64-contrib:
    This archive contains all the contributed programs
    compiled for the x86_64 architecture, for use with the
    x86_64 native version.
linux-ppc-contrib:
    Detta arkiv innehåller alla bidragande program
    kompilerade för PowerPC-arkitekturen, för användning
    med Linux/PowerPC portningen.

linux-ppc-sdk:
