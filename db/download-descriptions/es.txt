    Fuentes de la documentación.

docs-html:
    Documentación en formato HTML. Contiene todo lo que está disponible
    en el sitio web adaptado para la lectura en desconexión.
freebsd5-i386-system:
    La versión alojada en FreeBSD/i386 5.x. Fíjate que no
    andará en los sistemas FreeBSD 4.x.
freebsd-i386-system:
    La versión alojada en FreeBSD/i386 5.x. Fíjate que no
    andará en los sistemas FreeBSD 4.x.
linux-i386-system:
    La versión alojada en Linux/i386. Este es el puerto más
    completo y maduro, o sea que tú puedes ejecutar todos
    los programas y demos incluidos en él. Este archivo
    contiene todos los binarios del sistema AROS compilados
    del árbol de Subversion.
mingw32-x86_64-system:
    The hosted version for Microsoft Windows on x86_64 platform.
mingw32-i386-system:
    The hosted version for Microsoft Windows on i386 platform. Developed
    and compiled under Windows XP, but should work on all 32-bit Windows
    versions starting from Windows 98.
linux-x86_64-system:
    La versión alojada en Linux/x86_64. Este es el puerto más
    completo y maduro, o sea que tú puedes ejecutar todos
    los programas y demos incluidos en él. Este archivo
    contiene todos los binarios del sistema AROS compilados
    del árbol de Subversion.
linux-ppc-system:
    La versión alojada en Linux/PowerPC. Este archivo contiene todos
    los binarios del sistema AROS compilados del árbol de Subversion.
pc-i386-boot-floppy:
    La versión nativa para las computadoras PC AT basadas en i386
    y compatibles. Este archivo contiene la imagen del disquete
    de arranque que puedes rápidamente escribir a un disquete y
    arrancar AROS con él.
pc-i386-boot-iso:
    La versión nativa para las computadoras PC AT basadas en i386 y
    compatibles. Ésta es la imagen ISO de CDROM para arrancar,
    que contiene todo el sistema AROS y los programas contribuídos.
pc-x86_64-boot-iso:
    La versión nativa para las computadoras PC de 64-bit (amd64, x86_64).
    Ésta es la imagen ISO de CDROM de arranque, que 
    contiene todo el sistema AROS más los programas contribuídos.
pc-x86_64-smp-boot-iso:
    This is a highly experimental SMP build for pc-x86_64.
    Only recommended for Developers. This is a bootable CD-ROM ISO image.
raspi-armhf-system:
    The native version for ARMv6 based Raspberry Pi computers.
    Contains all files needed to boot from a FAT formatted sd-card.
raspi-armhf-contrib:
    This archive contains all the contributed programs,
    compiled for the ARM architecture, for use with the
    RasPi/ARM port (using hard-float ABI). Please note that this
    version requires at least an ARMv6 CPU with VFP unit.
raspi-armhf-ports:
    This archive contains applications and games which are built
    from external sources for the armhf architecture (both hosted
    and native). Because of its size the archive will only be updated
    from time to time on the snapshots page.
pc-i386-contrib:
    Este archivo contiene todos los programas contribuídos
    compilados para la arquitectura i386. Éstos funcionarán en
    la versión apropiada sea alojada (p.e. i386-linux) o nativa
    (i386-pc).
pc-x86_64-contrib:
    Este archivo contiene todos los programas contribuídos
    compilados para la arquitectura x86_64, para usar con la
    versión nativa x86_64.
linux-ppc-contrib:
    Este archivo contiene todos los programas contribuídos
    compilados para la arquitectura PowerPC, para usar con 
    el puerto Linux/PowerPC.
pc-i386-sdk:
    El conjunto de archivos y herramientas para construir tus
    propios programas AROS en tu máquina preferida. También
    necesitarás gcc y binutils configurados para producir
    ejecutables i386-elf.
linux-ppc-sdk:
    El conjunto de archivos y herramientas para construir tus
    propios programas AROS en tu máquina preferida. También
    necesitarás gcc y binutils configurados para producir
    ejecutables ppc-elf.
pc-x86_64-sdk:
    El conjunto de los archivos y herramientas para armar tus
    propios programas AROS en tu máquina preferida. También
    necesitarás gcc y binutils configurados para producir los
    ejecutables x86_64-elf.
