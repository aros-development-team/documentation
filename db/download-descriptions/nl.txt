freebsd5-i386-system:
    De gehoste versie voor FreeBSD/i386 5.x. Deze werkt overigens
    niet onder FreeBSD 4.x systemen.
freebsd-i386-system:
    De gehoste versie voor FreeBSD/i386 5.x. Deze werkt niet onder
    FreeBSD 4.x systemen.
linux-i386-system:
    De gehoste versie voor Linux/i386. Dit is de meest complete en
    capabele versie, wat betekend dat je hiermee alle bijgesloten
    programma's en demo's kunt draaien. Dit archief bevat ook alle AROS
    systeem binaries gecompileerd vanuit de Subversion tak. 
mingw32-x86_64-system:
    The hosted version for Microsoft Windows on x86_64 platform.
mingw32-i386-system:
    The hosted version for Microsoft Windows on i386 platform. Developed
    and compiled under Windows XP, but should work on all 32-bit Windows
    versions starting from Windows 98.
linux-i386-system:
    De gehoste versie voor Linux/x86_64. Dit is de meest complete en
    capabele versie, wat betekend dat je hiermee alle bijgesloten
    programma's en demo's kunt draaien. Dit archief bevat ook alle AROS
    systeem binaries gecompileerd vanuit de Subversion tak. 
linux-ppc-system:
    De gehoste versie voor Linux/PowerPC. Dit archief bevat alle AROS
    systeem binaries gecompileerd vanuit de Subversion tak.
pc-i386-boot-floppy:
    De native versie voor i368-gebaseerde PC AT compatibele
    computers. Dit archief bevat de boot floppy disk image, welke naar een
    schijf te schrijven is om AROS mee te starten. 
pc-i386-boot-iso:
    De native versie voor i386-gebaseerde PC AT compatibele computers.
    Dit is een opstartbaar CDROM ISO bestand, dat alle AROS systeem en
    gecontribueerde programma's bevat.
raspi-armhf-system:
    The native version for ARMv6 based Raspberry Pi computers.
    Contains all files needed to boot from a FAT formatted sd-card.
raspi-armhf-contrib:
    This archive contains all the contributed programs,
    compiled for the ARM architecture, for use with the
    RasPi/ARM port (using hard-float ABI). Please note that this
    version requires at least an ARMv6 CPU with VFP unit.
raspi-armhf-ports:
    This archive contains applications and games which are built
    from external sources for the armhf architecture (both hosted
    and native). Because of its size the archive will only be updated
    from time to time on the snapshots page.
pc-i386-contrib:
    Dit archief bevat alle gecontribueerde programma's gecompileerd voor
    de i386 architectuur. Deze werken zowel in de gehoste (bijv. i386-linux)
    als native (i386-pc) versies. 
pc-x86_64-boot-iso:
    De native versie voor 64-bit (x86_64, amd64) PC computers. Dit omvat een
    opstartbare CD-ROM ISO image, welke alle AROS systeem en contributie
    programmatuur bevat.
pc-x86_64-smp-boot-iso:
    This is a highly experimental SMP build for pc-x86_64.
    Only recommended for Developers. This is a bootable CD-ROM ISO image.
pc-x86_64-contrib:
    This archive contains all the contributed programs
    compiled for the x86_64 architecture, for use with the
    x86_64 native version.
linux-ppc-contrib:
    Dit archief bevat alle gecontribueerde programma's gecompileerd voor
    de PowerPC architectuur, voor gebruik met Linux/PowerPC port.
pc-i386-sdk:
    Een set van bestanden en tools waarmee u eigen AROS programma's kunt
    bouwen op een machine naar uw keuze. Benodigd zijn ook gcc en binutils,
    beiden geconfigureerd voor het produceren van i386-elf executables.
linux-ppc-sdk:
    Een set van bestanden en tools waarmee u eigen AROS programma's kunt
    bouwen op een machine naar uw keuze. Benodigd zijn ook gcc en binutils,
    beiden geconfigureerd voor het produceren van ppc-elf executables.
pc-x86_64-sdk:
    Een set van bestanden en tools waarmee u eigen AROS programma's kunt
    bouwen op een machine naar uw keuze. Benodigd zijn ook gcc en binutils,
    beiden geconfigureerd voor het produceren van x86_64-elf executables.
